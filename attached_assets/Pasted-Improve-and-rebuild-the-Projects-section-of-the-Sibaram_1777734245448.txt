=== COMPONENT ARCHITECTURE ===

components/Projects/
  index.tsx           (section wrapper, filter logic, grid)
  ProjectCard.tsx     (collapsed card view)
  ProjectExpanded.tsx (expanded panel content)
  ProjectFilter.tsx   (category filter bar)
  MetricBadge.tsx     (single metric pill)
  StatusChip.tsx      (completed / in-progress / archived chip)
  CategoryBadge.tsx   (pixel category badge, colored per type)

=== FILTER BAR ===

Horizontal pill buttons above the grid:
  [All] [CV] [ML] [Research] [Web] [Systems]

Active filter: filled with --accent color, 1px border.
Inactive: transparent background, 1px border.
Pixel press effect on click.
Filter animates cards out/in using Framer Motion layoutId.
"Featured only" toggle on the right side of the filter bar.
Filtering does not reload the page - pure client state.

=== CARD DESIGN (collapsed) ===

Card dimensions: fixed min-height 220px, full column width.
Border: 1px solid var(--border).
Background: var(--surface).
No border-radius - sharp corners only (pixel aesthetic).

Layout (top to bottom):

  ROW 1: CategoryBadge (left) + StatusChip (right)
    CategoryBadge: pixel 8x8 SVG icon + label text
      CV     -> cyan   (#00D4F5) icon
      ML     -> violet (#9B4FE8) icon
      Web    -> green  (#22C55E) icon
      Research -> amber (#F59E0B) icon
      Systems -> red   (#EF4444) icon
    StatusChip:
      completed    -> green border + text, no fill
      in-progress  -> amber border + text, blinking dot (CSS animation)
      archived     -> muted border + text

  ROW 2: Project title (JetBrains Mono, 18px, --text-primary)

  ROW 3: Tagline (Inter, 13px, --text-muted, max 2 lines, ellipsis overflow)

  ROW 4: Tech stack chips (PixelChip components, wrapping flex row)
    Max 4 chips visible, "+N more" chip if overflow

  ROW 5: Metrics row (MetricBadge components)
    Each metric: label in muted + value in accent, separated by pipe |
    Example: "Accuracy | 94.2%   Classes | 6   Dataset | 25k images"

  ROW 6: Footer row
    Left: [Live Demo] [GitHub] [Paper] - only render if link exists
      Each as pixel-bordered anchor, 12px monospace text
    Right: Year chip + expand arrow icon (Lucide ChevronDown)

Hover state on entire card:
  - 2px translate(-2px, -2px)
  - box-shadow: 4px 4px 0px var(--accent)
  - Transition: 150ms ease

=== EXPANDED PANEL ===

Slides open below the card using Framer Motion:
  initial: { height: 0, opacity: 0 }
  animate: { height: "auto", opacity: 1 }
  transition: { duration: 0.3, ease: "easeInOut" }
  
Expand arrow rotates 180deg when open (CSS transform transition).

Content inside expanded panel:
  Separator: 1px dashed var(--border), full width

  Three labeled blocks, each with a pixel icon prefix:

  [>] THE PROBLEM
      problem field text, 13px Inter, --text-muted

  [>] THE APPROACH
      approach field text, 13px Inter, --text-muted

  [>] THE OUTCOME
      outcome field text, 13px Inter, 
      if status is in-progress: outcome text + amber "Work in progress" tag

  Full tech stack (all chips, not truncated)
  All links repeated (Live Demo, GitHub, Paper)

Only one card can be expanded at a time.
Clicking an open card closes it.
Clicking a different card closes the previous one and opens the new one.

=== FEATURED ROW ===

Above the main grid, render featured: true projects in a horizontal 
scrollable row labeled "FEATURED WORK" with a pixel star icon.
This row shows on all filter states.
Each featured card in this row is wider (takes 60% width, min 320px).
Non-featured projects appear in the regular grid below.

=== PIXEL CATEGORY BADGE SPECS ===

Built from SVG <rect> elements only. Each is 8x8 pixel grid on a 24x24 viewBox.
Pixel size = 3px (24 / 8 = 3px per pixel).

CV badge (eye icon shape):
  Row 2: pixels 2,3,4,5,6 filled
  Row 3: pixels 1,3,5,7 filled (pupil gap)
  Row 4: pixels 2,3,4,5,6 filled

ML badge (brain/network icon):
  3 nodes connected by lines, abstract

Web badge (globe icon):
  Circle outline with horizontal lines

Research badge (flask icon):
  Simple test tube shape

Build each as its own SVG component with fill="currentColor" 
so it inherits the category color from the parent.

=== ANIMATIONS ===

Grid reflow on filter change:
  - Framer Motion AnimatePresence with mode="popLayout"
  - Exiting cards: { opacity: 0, scale: 0.95, transition: 0.15s }
  - Entering cards: { opacity: 0, y: 10 } -> { opacity: 1, y: 0 }
  - Stagger entering cards by 0.05s each

Section entrance (first load):
  - Cards animate in with 0.08s stagger, y: 20 -> 0, opacity 0 -> 1
  - Trigger via IntersectionObserver, animate once
