=== DESIGN SYSTEM ===

The aesthetic is: "calm pixel laboratory" - minimalist base with deliberate pixel art 
accents. NOT a pixel art website. A clean, modern site that uses pixel art as a design 
language in specific places. Think: a dark research lab where someone tacked hand-drawn 
pixel sprites on the walls.

LIGHT MODE palette:
  --bg: #F5F5F0
  --surface: #EBEBE6
  --border: #D0D0C8
  --text-primary: #1A1A1A
  --text-muted: #6B6B6B
  --accent: #00B4D8 (electric cyan)
  --accent-2: #7B2FBE (deep violet)
  --pixel-ink: #1A1A1A

DARK MODE palette:
  --bg: #0D0D10
  --surface: #141418
  --border: #252530
  --text-primary: #E8E8E0
  --text-muted: #9090A0
  --accent: #00D4F5
  --accent-2: #9B4FE8
  --pixel-ink: #E8E8E0

Rules:
- No gradients except as very subtle background texture
- Borders use 1px solid var(--border) everywhere
- Spacing based on 8px grid
- Every interactive element has a pixel-style :hover state 
  (2px offset box-shadow in --accent color, like an 8-bit button press)
- Scrollbar styled: thin, accent colored

PIXEL ACCENT RULES (inline SVG sprites - no external image files):
- Each section header has a small pixel art icon (8x8 or 16x16 SVG) 
  left of the title, hand-crafted to match section theme
- Hero: a tiny animated pixel character (CSS keyframe idle animation - 
  bobbing 2px up and down, 800ms loop)
- Skills section: pixel bar charts instead of plain progress bars
- Projects: pixel "badge" in top-left corner of each card showing category 
  (CV, ML, Web, etc.)
- Contact/Footer: pixel art of a CRT monitor with blinking cursor
- Dividers between sections: a single row of pixel dots (4x4 each, 8px gap, 
  muted color)

=== PIXEL ART SPECS (inline SVG only) ===

All pixel art is built from <rect> elements on a viewBox grid.
No external images. No raster files.
Each pixel = one <rect> element with fill="currentColor" or fill="var(--accent)".

Hero sprite: 16x24 grid, simple humanoid figure, idle bounce animation via CSS.
Category badges: 8x8, different color per category (CV=cyan, ML=violet, Web=green).
Section icons: 16x16, one per section, thematically matched.
CRT monitor: 32x24, simple monitor shape with screen glow effect.
Pixel divider: 1 row, repeating 4x4 dots.
