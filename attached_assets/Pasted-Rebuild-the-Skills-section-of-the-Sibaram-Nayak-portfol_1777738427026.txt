=== PIXEL BAR CHART (PixelBarChart.tsx) ===

Horizontal bar chart. One bar per category.
Shows skill count as a pixel-block bar.

Each row:
  Category label: 140px fixed width, 
    11px JetBrains Mono, --text-muted, right-aligned
  Bar: flex row of filled/empty pixel blocks
    Each block: 12x12px square, 3px gap between blocks
    Max blocks: 8 (normalize to largest category)
    Filled blocks: var(--accent), 1px solid var(--accent)
    Empty blocks: transparent, 1px solid var(--border)
  Count label: "7 skills" 10px --text-muted, left of bar

Hover on a row: accent color fills label, 
  clicking the row activates that category tab

Entrance animation:
  Bars fill left to right, block by block
  Each block: 60ms delay * block index
  Stagger categories 100ms apart
  Trigger once on IntersectionObserver

=== CATEGORY TABS (CategoryTabs.tsx) ===

Horizontal tab row, margin-top 32px.
One tab per category. Scrollable on mobile.

Each tab:
  Pixel category icon (16x16 SVG) + label text
  Font: 11px JetBrains Mono
  Padding: 8px 16px
  Border: 1px solid --border
  No border-radius
  Background: transparent

Active tab:

  Color: #0D0D10

  Icon: inverted color

Hover (inactive):

Tab switching:

  Panel content: AnimatePresence mode="wait"
    Exit: x: -10, opacity: 0, duration 150ms
    Enter: x: 10->0, opacity: 0->1, duration 200ms

  Border-color: var(--accent)
  Transform: translate(-2px, -2px)
  Box-shadow: 3px 3px 0px var(--accent)
  Skill name color: --accent
  Transition: 120ms ease

Chip entrance (when panel opens):
  Stagger 0.04s per chip
  y: 8 -> 0, opacity: 0 -> 1
  duration: 200ms
