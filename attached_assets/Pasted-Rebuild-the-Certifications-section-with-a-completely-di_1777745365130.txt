=== WALL LAYOUT CONCEPT ===

TheWall.tsx is a CSS grid of document slots.
Desktop: 4 columns, auto rows.
Tablet: 3 columns.
Mobile: 2 columns.

Each slot: 180px wide, 220px tall, position relative.

Documents inside slots are NOT axis-aligned.
Each PinnedDoc has a small random rotation:
  Completed certs: rotate between -3deg and +3deg
  In-progress: rotate between -6deg and +2deg
  Planned (sticky): rotate between -8deg and +8deg

Rotation is DETERMINISTIC not random:
  Derive from cert id string hash so it never changes
  between renders. No Math.random() - use a seeded
  function based on cert.id charCode sum % range.

  function getRotation(id: string, min: number, max: number) {
    const seed = id.split('').reduce(
      (acc, c) => acc + c.charCodeAt(0), 0
    )
    return min + (seed % (max - min))
  }

=== WALL ENTRANCE ANIMATION ===

Documents drop onto wall one by one:
  Each PinnedDoc/StickyNote:
    initial:  { y: -40, opacity: 0, rotate: 0 }
    animate:  { y: 0,   opacity: 1, rotate: Ndeg }
    
  Stagger: 0.08s per document
  Duration: 400ms per document
  Ease: "easeOut" with slight bounce
    (spring: stiffness 200, damping 15)

  Effect feels like documents being placed/dropped 
  onto the wall one by one.

Trigger: IntersectionObserver, animate once.
